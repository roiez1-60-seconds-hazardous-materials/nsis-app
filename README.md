# NSIS — Non-Targeted Sequencing Identification System

מערכת זיהוי רצף גנומי בלתי ממוקד

Interactive web application showcasing the NSIS biological identification system based on Nanopore Sequencing technology.

## Features

- 🧬 Interactive DNA helix animation
- ⚡ Real-time current signal visualization  
- 🎯 Step-by-step incident simulation
- 🌐 Bilingual support (Hebrew / English)
- 📚 Full bibliography (23 verified sources)
- 📱 Responsive design

## Tech Stack

- Next.js 15
- React 19
- Canvas API animations
- CSS-in-JS

## Deploy

Deployed via [Vercel](https://vercel.com).
